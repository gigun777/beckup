# SettingsWindowSystem (SWS) — Empty Modal Shell Demo

Цей демо-архів показує **ту саму модалку SWS**, як у проєкті (структура + CSS),
але **без будь-яких пунктів/карток/списків у тілі**.

## Що саме "порожнє"
- `.sws-body` створюється стандартно, але контент-функція повертає `wrap([])` — тобто **0 елементів**.

## Запуск
Відкрий `demo_empty.html` у браузері.

## API
Додано метод:
- `SettingsWindow.openEmptyRoot(opts)` — відкриває SWS і пушить **порожній** root-екран.
